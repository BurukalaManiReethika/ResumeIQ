# 🤖 ResumeIQ

AI-Powered Resume Analyzer built with ASP.NET Core, SQL Server, and JavaScript.

## ✨ Features

* Resume Upload
* ATS Score Analysis
* Skill Gap Detection
* Resume Insights
* Authentication
* Dashboard Analytics

## 🛠 Tech Stack

### Backend

* ASP.NET Core
* C#
* Entity Framework Core

### Frontend

* HTML
* CSS
* JavaScript

### Database

* SQL Server
