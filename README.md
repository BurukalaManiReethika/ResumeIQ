# 🤖 ResumeIQ

AI-Powered Resume Analyzer built with ASP.NET Core, SQL Server, and JavaScript.
  ![ASP.NET](https://img.shields.io/badge/ASP.NET-Core-blue)![SQL Server](https://img.shields.io/badge/SQL-Server-red)![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)![License](https://img.shields.io/badge/License-MIT-green)

## ✨ Features

* Resume Upload
* ATS Score Analysis
* Skill Gap Detection
* Resume Insights
* Authentication
* Dashboard Analytics

### Frontend

* JavaScript
